# Tic-Tac-Toe

A simple browser-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features

- 3x3 Tic-Tac-Toe board using clickable buttons
- Two-player turn-based gameplay (O and X)
- Win detection for all winning combinations
- Draw detection when the board is full
- Reset and New Game buttons to restart the board
- Styled responsive game layout

## Files

- `index.html` — game structure and UI elements
- `style.css` — board layout, buttons, and message styling
- `app.js` — game logic, event handling, win/draw checking, and reset behavior

## How to use

1. Open `index.html` in a web browser.
2. Click any empty square to place O or X.
3. The game alternates turns automatically.
4. When a player wins or the game is a draw, a message appears.
5. Click `New Game` or `Reset` to clear the board and play again.

## Notes

- The game uses a local JavaScript implementation with no backend.
- The button grid is fully interactive and disables squares once played.
- The game shows a congratulatory message for a winner and handles tie games.
